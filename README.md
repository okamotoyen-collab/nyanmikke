# にゃんみっけ！🐾

野良猫や地域猫を見つけた場所・写真を共有できる投稿アプリです。

## URL
https://nyanmikke.onrender.com/

## 使用技術
- Java
- Spring Boot
- Thymeleaf
- PostgreSQL
- Render
- Cloudinary
- HTML/CSS

## 機能
- 写真投稿
- タグ分類
- いいね機能
- 詳細ページ
- 投稿削除
- レスポンシブ対応

## 工夫した点
- スマホでも見やすいカードUI
- Cloudinaryを使った画像管理
- PostgreSQLによるデータ永続化
- タグで投稿を絞り込み可能

## 今後追加したい機能
- ログイン機能
- コメント機能
- 地図表示
- 通報機能
